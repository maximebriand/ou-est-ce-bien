Politique de confidentialité

OùEstCeBien — Extension de navigateur
Dernière mise à jour : 11 février 2026

OùEstCeBien est une extension de navigateur éditée par Maxime Briand.
Contact : contact@maximebriand.fr

1. Données collectées

OùEstCeBien ne collecte aucune donnée personnelle.

L'extension ne dispose d'aucun serveur backend. Aucune information n'est envoyée à l'éditeur. Il n'y a aucun système d'analytics, de tracking, ni de télémétrie.

2. Données traitées localement

Les données suivantes sont traitées exclusivement dans votre navigateur et stockées localement :

  • Données DPE extraites des annonces consultées (classe énergie, surface, code postal, date DPE)
  • Cache de propriétés pour accélérer les recherches ultérieures (max. 500 entrées, supprimées après 30 jours)
  • Compteur de recherches (version gratuite)
  • Clé de licence et identifiant d'instance (version Pro)

Ces données ne quittent jamais votre navigateur, sauf pour interroger les API publiques décrites ci-dessous.

3. API externes consultées

L'extension interroge exclusivement des API publiques françaises pour fournir ses fonctionnalités :

  • ADEME (data.ademe.fr) — Base nationale des Diagnostics de Performance Énergétique et des audits énergétiques
  • DVF / Opendatasoft (public.opendatasoft.com) — Demandes de Valeurs Foncières (transactions immobilières)
  • API Carto IGN (apicarto.ign.fr) — Données cadastrales
  • Bien'ici (bienici.com) — Enrichissement géographique des annonces

Les requêtes envoyées à ces API contiennent uniquement des critères de recherche techniques (classe énergie, surface, code postal, coordonnées GPS). Aucune donnée personnelle n'est transmise.

4. Paiement et licence

Le paiement de la version Pro est géré intégralement par Lemon Squeezy, un prestataire tiers. OùEstCeBien n'a accès à aucune donnée bancaire. Seules la clé de licence et l'adresse email associée sont échangées lors de l'activation.

La politique de confidentialité de Lemon Squeezy s'applique pour le traitement des paiements : lemonsqueezy.com/privacy

5. Permissions du navigateur

L'extension demande les permissions suivantes :

  • activeTab / tabs — Détecter les annonces immobilières sur l'onglet actif
  • sidePanel — Afficher le panneau latéral de résultats
  • storage — Stocker les préférences et le cache localement
  • alarms — Revalider périodiquement la licence Pro
  • Accès aux sites immobiliers — Extraire les données DPE des annonces (Leboncoin, SeLoger, PAP, Bien'ici, Logic-Immo, Century 21, Safti, IAD France, ParuVendu, Ouest France Immo, Notaires Immobilier)

6. Suppression des données

La désinstallation de l'extension supprime automatiquement toutes les données stockées localement. Vous pouvez également effacer les données manuellement via les paramètres de votre navigateur.

Pour désactiver votre licence Pro (libérer un emplacement d'appareil), utilisez le bouton de désactivation dans l'extension avant de la désinstaller.

7. Modifications

Cette politique peut être mise à jour. La date de dernière mise à jour est indiquée en haut du document. En cas de changement substantiel, une notification sera affichée dans l'extension.
